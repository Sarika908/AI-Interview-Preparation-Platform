# 🤖 AI Interview Preparation Platform

An AI-powered Interview Preparation Platform built using Django and Google Gemini AI.

## 🚀 Features

- 🔐 User Authentication
- 📄 AI Resume Analyzer
- 🤖 AI Mock Interview
- 💻 Coding Assessment
- 📝 Technical MCQ Quiz
- 📊 Dashboard & Analytics
- 📄 PDF Interview Report
- 🎤 Voice Interview
- 🌐 Responsive Bootstrap UI

## 🛠 Tech Stack

- Python
- Django
- SQLite
- Bootstrap 5
- Google Gemini AI
- HTML
- CSS
- JavaScript
- ReportLab

## Installation

```bash
git clone https://github.com/yourusername/AI-Interview-Preparation-Platform.git

cd AI-Interview-Preparation-Platform

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
